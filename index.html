Catany: Catan Board Generator



Catany is a Catan Board Generator was one of my first projects, that builds a board for the game Settlers of Catan. It allows players to generate a fully randomized board or choose options to build a board for Catan players to play the game in new unique ways.

SimpleSport



SimpleSport is a sports team management web application I built for the Capstone Project of my degree. I used popular technologies such as React and Express in order to create it to expand my skill set.
